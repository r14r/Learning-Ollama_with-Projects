# Pflichtenheft: Model Info

## Expected Functionality
This script demonstrates how to retrieve and display information about Ollama models, including listing available models and getting detailed model information.

## Input
- **Function parameters**:
  - `model_name` (str): Name of a specific model to query
  - No user input required for basic execution

## Expected Output
```
=== Ollama Model Information ===

Available Models:
--------------------------------------------------------------------------------
Name: llama3:latest
Size: 4.66 GB
Modified: 2024-01-08T10:30:00Z
--------------------------------------------------------------------------------

Detailed Information for 'llama3:latest':
================================================================================

Modelfile:
# Modelfile generated by "ollama show"
FROM /path/to/model
PARAMETER temperature 0.8
...

Parameters:
stop                   "<|start_header_id|>"
stop                   "<|end_header_id|>"
...

Template:
{{ if .System }}{{ .System }}{{ end }}{{ .Prompt }}
```

## Tests

### Test 1: List Models
**Input:** `list_models()`  
**Expected Output:** List of dictionaries with model information (name, size, modified_at)

### Test 2: Show Model Info
**Input:** `show_model_info("llama3")`  
**Expected Output:** Dictionary with detailed model information

### Test 3: Format Size
**Input:** `format_size(4664729600)`  
**Expected Output:** "4.34 GB"

### Test 4: Handle No Models
**Input:** `list_models()` when no models are installed  
**Expected Output:** Empty list []

### Test 5: Invalid Model
**Input:** `show_model_info("nonexistent_model")`  
**Expected Output:** Empty dictionary {} with error message

## Dependencies
```
ollama>=0.1.0
```

## Prerequisites
- Ollama must be installed and running
- At least one model should be pulled for meaningful output

## Usage
```bash
# Ensure at least one model is available
ollama pull llama3

# Run the script
python script.py
```

## Learning Objectives
- Use the ollama.list() function
- Use the ollama.show() function
- Parse model metadata
- Format and display information
- Handle cases with no models
